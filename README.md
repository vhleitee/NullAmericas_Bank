# NullAmericas Bank

## 📌 Sobre o Projeto
O **NullAmericas Bank** é um sistema de aplicação bancária desenvolvido como projeto prático para a avaliação **AV2** da faculdade, sob a orientação do **Professor Everton Pereira**.

O principal objetivo do projeto é consolidar conceitos de:
* **Programação Orientada a Objetos (POO)**
* **Persistência de dados** em ambiente relacional
* **Desenvolvimento de interfaces visuais**

O sistema simula as operações essenciais e o fluxo de segurança de uma instituição financeira real.

---

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Banco de Dados:** MySQL (para persistência de usuários, contas e histórico de transações)
* **Interface Gráfica:** Interface Java (Desktop) para interação amigável com o usuário
* **Controle de Versão:** Git e GitHub

---

## 🚀 Principais Funcionalidades (Escopo do Sistema)
* **Gerenciamento de Usuários e Contas:** Criação, autenticação (login seguro) e manutenção de contas bancárias.
* **Operações Financeiras:** Realização de depósitos, saques e transferências entre contas.
* **Conexão em Tempo Real:** Comunicação direta com o banco de dados MySQL para atualização instantânea de saldos e registros.
* **Histórico / Extrato:** Consulta de transações realizadas para maior controle e transparência do usuário.
